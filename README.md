# HealthIndeX
HealthindeX is a module for calculating health and medical indices

	Body Mass Index (BMI)
	Body Build Index (BBI)
	Body Adipose Index (BAI)
	Corpulence Index (CI)


### usage:
using HealthIndeX

##### calculating body mass index
     + First Method
          bodymassindex(mass,height)
  
      + Short Method
          bmi(mass,height)
